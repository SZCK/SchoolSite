### 使用说明
本作品不仅有静态页面，还提供了可选的PHP动态页面。
##### 目录结构：
- /static/ - 静态文件目录(css,js,font...)
- /cgi/ - 动态文件目录(php)
- /s-data/ - 静态模式的演示数据(json)
- index.html - 主页文件
- config.json - 配置文件
### 静态/动态模式说明
###### 静态模式
自动从/s-data/目录调取数据json文件并且解析。
###### 动态模式
自动从/cgi/目录调取php脚本，解析脚本返回的json。
#### 切换模式
修改config.json，里面“cgi”值为1，则开启动态模式，为0则开启静态模式。
